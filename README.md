🌸 Авторассылка сообщений в чаты услуг на Python 🌸

Твой персональный помощник для автоматических рассылок по группам и каналам Telegram

✨ Что умеет бот:

• 👤 Добавлять и хранить твои аккаунты

• 📋 Добавлять группы вручную (@username), по ссылке (t.me/...), из .txt файла или сканировать все группы аккаунта

• 🔗 Подписываться на группы списком

• 📨 Автоматически рассылать сообщения с любым интервалом (постоянным или случайным)

• 📝 Использовать готовые шаблоны сообщений (до 50 штук)

• 🔒 Автоматически добавлять гаранта (например @username) в конец каждого сообщения

• 📊 Смотреть историю последних 10 отправок

• 👥 У каждого пользователя свои данные — никто не видит чужие аккаунты и группы
